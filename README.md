<h3>Installation</h3>
Clone the repository, add execute privileges, and run the script as described below:

`git clone https://github.com/Bill-Joe-Tech/FixMyCorruptPacman && cd FixMyCorruptPacman && chmod +x fixcorruptdatabase.sh && ./fixcorruptdatabase.sh`


<h3>Uninstallation</h3>
Remove the respective folder:

`rm -rv FixMyCorruptPacman`
